How to install TeamsPlus

A comprehensive guide on how to install TeamsPlus.

  1. Download the extension

    Download the .zip file from the website (this website) or from GitHub.

  2. Open the downloaded file

    Open the .zip file that you have downloaded. It should be in your Downloads folder on Windows.

  3. Extract the file

    Once opened, press Extract All at the top of the window (File Explorer). If prompted to choose a path to extract to, just continue with the default path and click Extract.

  4. Open the browser extensions page
    • Open your favorite browser
    • Navigate to chrome://extensions (or edge://extensions on Microsoft Edge)
  5. Enable developer mode

    In the top right of the page (or middle left for Edge), toggle Developer mode on

  6. Load the extension
    • Click Load unpacked
    • Select the folder you extracted in step 3
  7. Confirm installation

    The extension should now appear in the extensions list and be active.

TeamsPlus

Browser extension for Microsoft Teams

Features

Custom themes

Modify the visual appearance of Microsoft Teams to better fit your preferences or workflow. Set your own background and foreground colors, adjust icon colors, and fine-tune the overall layout to reflect a setup that works best for you.

Games

Access a selection of simple, built-in games directly within Microsoft Teams. These are available instantly with no installations or additional permissions, designed for brief moments of downtime or quiet collaboration.

Custom fonts

Select and use fonts from external sources to adjust the text appearance in Teams. This can help with readability, accessibility, or simply achieving a consistent style across different documents and discussions.

Encryption

Protect conversations and shared content with message encryption. This minimizes the risk of exposure in the event of unauthorized access and supports a safer communication environment for both personal and professional use.
